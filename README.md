# Usage:

```
git clone git@github.com:edvancium/edvancium.github.io.git
cd edvancium.github.io
virtualenv .venv
source .venv/bin/activate
pip install -r requirements.txt
python upodate.py
git commit -a -m "site update"
git push
```
